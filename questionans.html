Q: Discuss the scope of var, let, and const

A: There are 3 kinds of scope in js(block scope,function scope,global scope). var is a funtion scope and other two let and const are block scope.

Q: Tell us the use cases of null and undefined

A: There are many use cases of null and undefined. Usually when we dont set any value it shows us undefined and when we know there will be no value we can set there null.

Q: What do you mean by REST API?

A: A REST API (also known as RESTful API) is an application programming interface (API or web API) that conforms to the constraints of REST architectural style and allows for interaction with RESTful web services.(collected from google)
